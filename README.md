# 📊 Sales Performance Dashboard using Matplotlib

A Python data visualization project that demonstrates how to create multiple charts in a single dashboard using the **Matplotlib** library. The dashboard provides insights into sales performance, product category revenue, advertising impact on sales, and regional sales distribution.

## 📌 Project Overview

This project is designed to visualize sales data using different chart types in a single dashboard. It helps users analyze sales trends, compare product performance, understand the relationship between advertising expenditure and sales, and identify regional sales contributions.

The dashboard contains four different visualizations:

- 📈 Monthly Sales Trend (Line Chart)
- 📊 Product Category Revenue (Bar Chart)
- 🔵 Ad Spend vs Sales (Scatter Plot)
- 🥧 Regional Sales Distribution (Pie Chart)

---

## ✨ Features

- Multiple charts in one dashboard
- Easy-to-understand visualizations
- Custom colors, titles, labels, and legends
- Grid layout using Matplotlib subplots
- Suitable for beginners learning Python Data Visualization

## 📊 Charts Included

### 1. Monthly Sales (Line Chart)

Shows monthly sales comparison between **Sales A** and **Sales B** from January to June.

**Purpose**
- Compare sales growth
- Identify monthly trends
- Observe performance differences

---

### 2. Product Category Comparison (Bar Chart)

Displays revenue generated from different product categories.

Categories:
- Electronics
- Clothing
- Food
- Furniture

**Purpose**
- Compare category performance
- Identify highest revenue category

---

### 3. Ad Spend vs Sales (Scatter Plot)

Shows the relationship between advertising spend and sales.

**Purpose**
- Analyze correlation between advertising and sales
- Understand business growth patterns

---

### 4. Regional Sales Distribution (Pie Chart)

Displays percentage contribution of different regions.

Regions:
- North
- South
- East
- West

**Purpose**
- Understand regional market share
- Compare sales distribution
